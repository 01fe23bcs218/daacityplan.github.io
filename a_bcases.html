IN TNC: TO FIND SHORTEST PATH

When a passenger requests a ride, the system can use Dijkstra’s Algorithm to compute the shortest path from the passenger’s location to the nearest available driver and also to find the shortest path to the passenger's destination. This helps optimize the time taken to reach the passenger, ensuring a quicker and more efficient ride-matching process.
